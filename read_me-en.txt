Very simple counter module for xoops2, and this module tested only xoops2.

	http://xoops-modules.sourceforge.jp/
	http://sourceforge.jp/projects/xoops-modules/
	
¢£Install
	Copy simplecounter directory to your "<XOOPS_DIR>/modules/".
	Install module from your admin control panel.
	Go to block admin and activate Simple Counter block.
	
¢£Update 
	from before 1.1.3 to 1.2/1.2.1
	You will need to point your browser to following url.
	http://<XOOPS_URL>/modules/simplecounter/admin/update_before_1_1_3_to_1_2.php
